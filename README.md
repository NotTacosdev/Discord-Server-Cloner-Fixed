*A Discord Server Cloner Which Can Clone Any Discord Server In Just Few Minutes. Made by SakshL and NotTacos, only works on WINDOWS!*
- Clone Channels
- Channel Permissions
- Server Roles
- Server Name
- Server Banner / Avatar

- Server Settings
